# 中文RSS订阅源精选

*中文 | [English](README_EN.md)*

精心整理的高质量中文RSS订阅源合集，涵盖科技、AI、网络小说、旅游等主题。

## 📋 项目简介

本项目收集和整理了优质的中文RSS订阅源，方便使用Reeder等RSS阅读器的用户快速导入和订阅。所有订阅源均以OPML格式组织，可直接导入主流RSS阅读器。

### 质量标准

- ✅ 更新频率稳定
- ✅ 内容原创性高
- ✅ 专业深度或实用价值
- ✅ 中文内容为主

## 🚀 使用方法

### 方法一：通过URL直接导入（推荐）

适用于支持URL导入的RSS阅读器（如Inoreader、Feedly等）：

1. 打开你的RSS阅读器
2. 找到"导入OPML"或"Import"功能
3. 选择"URL"或"From URL"选项
4. 粘贴以下URL：
   ```
   https://raw.githubusercontent.com/andybowu/rss/main/chinese-rss-feeds.opml
   ```
5. 点击导入即可

### 方法二：下载文件后导入

适用于所有RSS阅读器（如Reeder、NetNewsWire等）：

1. 下载 `chinese-rss-feeds.opml` 文件
2. 打开你的RSS阅读器
3. 找到"导入OPML"或"Import"功能
4. 选择"Device"或"从设备"选项
5. 选择下载的OPML文件导入

### 订阅特定网络小说

对于网络小说订阅，需要通过RSSHub：

**起点中文网**：
- 订阅作者：`https://rsshub.app/qidian/author/{作者ID}`
- 小说讨论区：`https://rsshub.app/qidian/forum/{小说ID}`
- 限时免费：`https://rsshub.app/qidian/free`

**晋江文学城**：
- 作品更新：`https://rsshub.app/jjwxc/book/{作品ID}`
- 作者作品：`https://rsshub.app/jjwxc/author/{作者ID}`

*注：可在作品或作者页面的URL中找到对应的ID*
