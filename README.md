## üöÄ How to Use

### Method 1: Import via URL (Recommended)

For RSS readers that support URL import (e.g., Inoreader, Feedly):

1. Open your RSS reader
2. Find the "Import OPML" or "Import" feature
3. Select "URL" or "From URL" option
4. Paste the following URL:
   ```
   https://raw.githubusercontent.com/andybowu/rss/main/chinese-rss-feeds.opml
   ```
5. Click import

### Method 2: Import Downloaded File

For all RSS readers (e.g., Reeder, NetNewsWire):

1. Download the `chinese-rss-feeds.opml` file
2. Open your RSS reader
3. Find the "Import OPML" or "Import" feature
4. Select "Device" or "From Device" option
5. Select the downloaded OPML file to import

### Subscribe to Specific Web Novels

For web novel subscriptions, use RSSHub:

**Qidian**:
- Subscribe to author: `https://rsshub.app/qidian/author/{author_id}`
- Novel forum: `https://rsshub.app/qidian/forum/{novel_id}`
- Limited-time free: `https://rsshub.app/qidian/free`

**Jinjiang Literature City**:
- Novel updates: `https://rsshub.app/jjwxc/book/{book_id}`
- Author works: `https://rsshub.app/jjwxc/author/{author_id}`

*Note: You can find the corresponding ID in the URL of the work or author page*
